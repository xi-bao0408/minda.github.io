HTML进阶篇-高级标签上

  • 作者： 喜宝
  • 时间： 2018-10-12
  • 分类： html、笔记 

<!DOCTYPE html>

这个标签现在几乎每个网页都会有，此为HTML5仅有的一个声明，它必须放在文件的开头在<html>之前。

HTML 4.01有三个声明

HTML 4.01 Strict
该 DTD 包含所有 HTML 元素和属性，但不包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" 
"http://www.w3.org/TR/html4/strict.dtd">

HTML 4.01 Transitional
该 DTD 包含所有 HTML 元素和属性，包括展示性的和弃用的元素（比如 font）。不允许框架集（Framesets）。

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" 
"http://www.w3.org/TR/html4/loose.dtd">

HTML 4.01 Frameset
该 DTD 等同于 HTML 4.01 Transitional，但允许框架集内容。

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" 
"http://www.w3.org/TR/html4/frameset.dtd">
<style type="text/css">

此标签是用来在HTML文件中使用css的一种格式

在它下面可以使用以下代码，来对<body>中的标签进行装饰

*{
	margin:0;
	padding:0;
  }

利用*{}来选择所有标签并利用margin:0;和padding:0;来初识化文章中的所有标签（以后会将CSS的内容）

&  ;

&为HTML中的编码特殊符（网页中的代码有很多需要特殊符来显示出来，而不是去执行）

A<br>B
它就等价于
	A
	B

<br>表示换行（他是一个单标签）

<hr>

此标签使用来在页面中显示一条水平线，就比如下面这条线

<ol  type=”1”   reversed=”reversed”   start=”2” > 
	<li></li>
	<li></li>
	<li></li>
</ol>

<ol>标签为有序列表
它里面的<li>为有序列表的列表架（用来有序存放东西）
<ol>中的type="1" 表示按数字顺序排序
reversed表示按倒序来排的
start=2"表示从第几个开始排（填数字就好）
下面为几组演示

<ol>		 <ol start="50">	<ol reversed>			<ol type="I">
  <li>咖啡</li>	   <li>咖啡</li>	  <li>咖啡</li>			  <li>咖啡</li>
  <li>牛奶</li>	   <li>牛奶</li>	  <li>牛奶</li>			  <li>牛奶</li>
  <li>茶</li>	   <li>茶</li>		  <li>茶</li>			  <li>茶</li>
</ol>		 </ol>			</ol>				</ol>
1.咖啡		50.咖啡		3.咖啡		I.咖啡
2.牛奶		51.牛奶		2.牛奶		II.牛奶
